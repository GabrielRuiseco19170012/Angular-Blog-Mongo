Project Description

Back End

This blog is a project developed to implement multiple relational and nonrelational databases on a server and developed by students of the Technological University of Torre√≥n.

We have used adonisJs as a framework for server development, adding mongodb as a Non-SQL database and mysql for the SQL part. Publications are directed to the SQL database and comments to Non-SQL database.

While adonisJS provides the Lucid models to drive the flow of data to the sql database, we have had to add the mongoose library to handle the data directed to the non-sql database.

Front End

In the Front End, we use the framwork Angular to create components for the different views that the blog handles, developing as well as the responsive client in a single page application(SPA).
